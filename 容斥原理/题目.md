## 题目一

[2513. 最小化两个数组中的最大值 - 力扣（LeetCode）](https://leetcode.cn/problems/minimize-the-maximum-of-two-arrays/description/)

二分答案，容斥原理判断

  ✅ 只能给 arr1 的数字：x / d2 - x / lcm

✅ 只能给 arr2 的数字：x / d1 - x/ lcm

✅ arr1/arr2 通用数字：x - a - b - x / lcm

```java
class Solution {
    public int minimizeSet(int d1, int d2, int c1, int c2) {
        long lcm = lcm(d1,d2);
        long res = -1;long l = 1,r = 4_000_000_009L;
        while(l <= r){
            long mid = (l + r) >> 1;
            boolean f = true;
            long a = mid / d2 - mid / lcm;
            long b = mid / d1 - mid / lcm;
            long c = mid - a - b - mid / lcm;
            if(a < c1){
                long x = c1 - a;
                if(c >= x)c -= x;
                else f = false;
            }
            if(b < c2){
                long x = c2 - b;
                if(c >= x)c -= x;
                else f = false;
            }
            if(f){
                res = mid;r = mid- 1;
            }else 
                l = mid + 1;
        }return (int)res;
    }
    public long lcm(long a,long b){
        return a / gcd(a,b) * b;
    }
    public long gcd(long a,long b){
        if(b == 0)return a;
        return gcd(b,a % b);
    }
}
```
