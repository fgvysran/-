# ## 题目一

[Problem - C - Codeforces](https://codeforces.com/contest/2156/problem/C)

思路：

1. 枚举最大公约数g

2. 大于等于4*g的数一定可以拆分为g的倍数.  类似:  g   (g + r)  xg （x >= 2):例如5g + 7 == g + (g + 7) + 3g

3. 然后除g,2g,3g以外的数都是坏数。

```java
import java.io.*;
import java.util.*;

public class Main {
    static BufferedReader br;
    static PrintWriter pw;
    static final long MOD = 998224353;
    static final int MOD2 = 251024;

    public static void main(String[] args) throws IOException {
        br = new BufferedReader(new InputStreamReader(System.in));
        pw = new PrintWriter(new BufferedWriter(new OutputStreamWriter(System.out)));
        //

        int t = Integer.parseInt(br.readLine());
        while(t-- > 0){
            solve();
        }

        //
        pw.flush();
        pw.close();br.close();
    }
    public static void solve() throws IOException {
        StringTokenizer st = new StringTokenizer(br.readLine());
        int n = Integer.parseInt(st.nextToken());
        int k = Integer.parseInt(st.nextToken());
        int[]map = new int[n + 1],sum = new int[n + 1];
        st = new StringTokenizer(br.readLine());
        for(int i = 1; i <= n; i++){
            int x = Integer.parseInt(st.nextToken());
            map[x] += 1;
        }
        for(int i = 1; i <= n; i++){
            sum[i] = sum[i - 1] + map[i];
        }int ans = 1;
        for(int g = 1; g <= n; g++){
            int h = Math.min(n,4 * g - 1);//小于等于h的都是坏数,>= 4 *g都可以转换为好数
            int good = n - sum[h];//好数
            good += map[g];//g的倍数是好数
            if(g * 2 <= n)good += map[g * 2];
            if(g * 3 <= n)good += map[g * 3];
            if(good >= n - k)ans = g;
        }
        pw.println(ans);
    }
}
```
